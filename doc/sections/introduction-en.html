Introduction back to ToC

The REPRODUCE-ME is an OWL2 ontology to describe a complete path of a scientific experiment. It links the computational and non-computational steps and data of scientific experiments. It extends the W3C PROV-O and the P-Plan ontology. It provides a set of classes and properties to represent a complete path of a scientific experiment to track the provenance of results. We extend the Entity, Agent, and Activity classes from PROV-O and Plan, Step, and Variable classes from P-Plan to describe several elements required for the reproducibility of scientific experiments. It describes a complete path of a scientific experiment taking into focus the use-case of light microscopy imaging experiments.
